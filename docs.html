Fusion Bot Documentation

Your go-to bot for moderation and fun.

/weather

Get weather data on a specific city, must provide a valid city name and country code.

/server-info

Get detailed information about the server where the command is ran in.

/user-info

Get detailed information on a user or yourself and their server roles.

/bot-settings

Edit and view Fusion's settings on your server such as moderation role, premium status, and the ability of the /help command.

/ban

Action a ban against a user in your server. The command runner must have the moderation role and cannot ban users with ADMINISTRATOR or BAN_MEMBERS permissions.
(Must setup a mododeration role ID with /bot-settings command).
